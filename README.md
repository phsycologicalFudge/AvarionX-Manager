AVarionX Manager is the official control centre for AVarionX Security (formerly known as CS Security) and CS Secure Files.

The manager enables:
- Automatic updates via Shizuku
- Access to advanced and system-level features for supported apps

## AVarionX Security

Repository:  
https://github.com/phsycologicalFudge/ColourSwift_AV

With Shizuku support (from version 3.0.2), users can enable System Watcher, an advanced version of the real-time protection module.
